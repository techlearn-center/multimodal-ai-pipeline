## 1. Setting Up Multimodal RAG

```python
from src.pipeline.multimodal_rag import MultimodalRAG

rag = MultimodalRAG(chroma_host="localhost", chroma_port=8001)

# Add text documents
rag.add_documents(
    texts=["The Eiffel Tower is 330 meters tall.", "Neural networks have layers of nodes."],
    metadatas=[{"type": "text", "source": "facts.txt"}, {"type": "text", "source": "ml.txt"}],
)

# Add image descriptions (generated by ImageProcessor)
rag.add_image_descriptions(
    image_paths=["data/images/tower.jpg"],
    descriptions=["Photo of the Eiffel Tower at sunset with golden lighting."],
)

# Query across all modalities
answer = rag.answer("How tall is the Eiffel Tower?")
print(answer)
```

## 2. Full Document Ingestion Pipeline

```python
from src.processors.document_processor import DocumentProcessor

processor = DocumentProcessor()
result = processor.process_pdf("data/documents/report.pdf")
full_text = "\n".join([p["text"] for p in result["pages"]])
chunks = processor.chunk_document(full_text, chunk_size=1000, overlap=200)

rag.add_documents(
    texts=[c["text"] for c in chunks],
    metadatas=[{"type": "text", "source": result["filename"], "chunk_id": c["chunk_id"]} for c in chunks],
    ids=[f"{result['filename']}_chunk_{c['chunk_id']}" for c in chunks],
)
```

## 3. Retrieval with Filters

```python
# Filter by content type
image_results = rag.query("What buildings are shown?", filter_type="image")
text_results = rag.query("What is the revenue?", filter_type="text")

# Control number of results
precise_results = rag.query("specific question", n_results=3)
broad_results = rag.query("general topic", n_results=10)
```

---

## 4. RAG Architecture

```
User Query
    |
    v
Embedding Model --> Query Vector
    |
    v
ChromaDB Search (with optional filters)
    |
    v
Top-K Results (text + image descriptions + metadata)
    |
    v
LLM (GPT-4o) with Context --> Answer with Citations
```
